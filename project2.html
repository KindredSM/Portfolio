Legendary Security

Front End / Web Design / Logo Design

Figma, HTML, CSS, GSAP, JavaScript

Live site

This website and set of designs were made for a security agency based in Brighton, UK.

The Process

A discussion with a colleague of mine at Legendary Security lead to a brainstorming session on how to build a professional presense for the business as the director had recently been looking to expand and compete with the larger agencies in the county.

Designing the site, logo and branding from scratch using figma, I took inspiration from other businesses in the area to give a familiar feel and formulate content in a way that is relevant to the goals of the company. When the design process was complete for the website utilising HTML, CSS, JS and Intersection Observer API, I built out the site one-to-one whilst referencing the design. I then linked up the clients custom domain and hosted the site on GitHub Pages.

Legendary security's new branding can be seen on their uniform worn by security professionals all over Brighton & Hove.
