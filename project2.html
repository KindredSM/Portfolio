Legendary Security

Front end / Web Design / Logo Design

Figma, HTML, CSS, JavaScript

Live site

This website was built for a security agency based in Brighton, UK.

The Process

A discussion with a colleague of mine at Legendary Security lead to a brainstorming session on how to build an online presense for the business as the founder had recently been looking to expand and compete with the larger agencies in the county.

Designing the site from scratch using figma, I took inspiration from other businesses in the area to give a familiar feel and formulate content in a way that is relevant to the goals of the site. When the design process was complete, utilising HTML, CSS, JS and Intersection Observer API I built out the site one-to-one whilst referencing the design. I then linked up the clients custom domain and hosted the site on GitHub Pages.
